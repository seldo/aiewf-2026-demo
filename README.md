# aiewf-2026-demo

Demos for AI Engineer World's Fair 2026 — the "Wonder Toys" shopping agent built
on the OpenAI Agents SDK (Next.js frontend + Python/FastAPI backend).

## Getting started

Head to **[`openai-agents-py/`](openai-agents-py/)** and follow its
[README](openai-agents-py/README.md) to get the app running locally.

## What's here

| Path | Description |
|------|-------------|
| [`openai-agents-py/`](openai-agents-py/) | The Wonder Toys agent (OpenAI Agents SDK, no observability). **Start here.** |
| [`openai-agents-py-pre-run/`](openai-agents-py-pre-run/) | Same app instrumented with Arize AX tracing |
| [`product-images/`](product-images/) | Shared product images, symlinked into each app's `public/` |
